JS-Tic-Tac-Toe
==============
Microverse Tic-Tac-Toe implemented with JS factory functions and inheritance pattern design

Have fun losing hours playing on our Tic-Tac-Toe gameboard the times you want. Register your name and go.

### Built with

- JS
- CSS/Bulma
- HTML
